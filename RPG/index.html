Jeu en réseau local uniquement

Ce jeu utilise des sockets TCP pour connecter deux machines sur le même réseau. Il ne peut pas tourner directement dans le navigateur — lancez-le avec Python sur deux ordinateurs du même réseau Wi-Fi ou filaire.

Lancer le jeu

  1. Assurez-vous que Python et Pyxel sont installés sur les deux machines : pip install pyxel
  2. Sur la machine Joueur 1 (serveur), ouvrez un terminal dans le dossier RPG/ et lancez : python main.py
    Le terminal affichera l'adresse IP locale (ex. 192.168.1.42).
  3. Sur la machine Joueur 2 (client), dans le même dossier, lancez avec l'IP du serveur : python main.py client 192.168.1.42
  4. Les deux fenêtres apparaissent dès que la connexion est établie. Bonne partie !

Contrôles

Joueur 1 — Serveur (bleu)

  • W Haut
  • S Bas
  • A Gauche
  • D Droite

Joueur 2 — Client (cyan)

  • ↑ Haut
  • ↓ Bas
  • ← Gauche
  • → Droite
