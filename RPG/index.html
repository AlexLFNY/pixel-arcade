Jeu en réseau local uniquement

Ce jeu nécessite deux machines sur le même réseau. Le Joueur 1 lance le serveur, le Joueur 2 s'y connecte avec l'IP indiquée. Il ne peut pas tourner dans le navigateur.

Lancer le jeu

  1. Installer Python et Pyxel sur les deux machines : pip install pyxel
  2. Machine Joueur 1 — ouvrir un terminal dans RPG/ et lancer :
    python main.py
    L'IP locale s'affiche dans le terminal (ex. 192.168.1.42).
  3. Machine Joueur 2 — même dossier, lancer avec l'IP du serveur :
    python main.py client 192.168.1.42
  4. Les deux fenêtres démarrent automatiquement dès la connexion établie.

Contrôles

Joueur 1 — Serveur (bleu)

  • W Haut
  • S Bas
  • A Gauche
  • D Droite

Joueur 2 — Client (cyan)

  • ↑ Haut
  • ↓ Bas
  • ← Gauche
  • → Droite

Mécaniques

  • 🍎 Ramassez des pommes pour gagner des vies (+1 à chaque collecte)
  • 🟦 Les deux joueurs bloquent mutuellement leurs déplacements (collision AABB)
  • ⬜ Les obstacles sont générés aléatoirement et partagés entre les deux écrans
  • ⏱️ Une pomme non ramassée se replace aléatoirement toutes les 10 secondes
  • 💀 Partie terminée quand les vies tombent à 0 — écran GAME OVER
